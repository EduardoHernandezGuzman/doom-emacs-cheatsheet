# Doom Emacs – Mis atajos esenciales

> Mi bitácora personal de comandos para Doom Emacs.

---

##  Archivos

- `SPC f f` → Buscar / abrir archivo
- `SPC f r` → Archivos recientes
- `SPC f s` → Guardar archivo
- `SPC f S` → Guardar todos los archivos
- `SPC f p` → Abrir `config.el`

---

##  Ventanas

- `SPC w v` → Dividir vertical
- `SPC w s` → Dividir horizontal
- `SPC w d` → Cerrar ventana
- `SPC w w` → Cambiar entre ventanas
- `SPC w m` → Maximizar ventana actual

---

##  Buffers

- `SPC b b` → Cambiar buffer
- `SPC b d` → Cerrar buffer actual
- `SPC b n` → Siguiente buffer
- `SPC b p` → Buffer anterior

---

##  Búsqueda

- `SPC /` → Buscar en proyecto
- `SPC s s` → Buscar en buffer actual
- `SPC s p` → Buscar en proyecto (ripgrep)
- `SPC *` → Buscar palabra bajo cursor

---

##  Git (Magit)

- `SPC g g` → Magit status
- `SPC g b` → Git blame
- `SPC g l` → Git log

---

##  LSP

- `SPC c d` → Ir a definición
- `SPC c r` → Renombrar símbolo
- `SPC c a` → Code action
- `SPC c f` → Formatear código

---

##  Movimiento (evil)

- `gg` → Inicio del archivo
- `G` → Fin del archivo
- `w` / `b` → Siguiente / anterior palabra
- `C-d` / `C-u` → Media página abajo / arriba
- `zz` → Centrar cursor

---

##  Mis ultra favoritos

- `SPC /` → Buscar en proyecto
- `SPC ,` → Cambiar buffer rápido
- `SPC g g` → Magit
- `SPC :` → `M-x` rápido
- `C-g` → Cancelar todo (botón del pánico)
